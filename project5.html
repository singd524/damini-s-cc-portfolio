Assignment #5 — Optical Illusion

Project Overview

I was inspired by kaleidoscopes, aiming to create complex symmetrical patterns that give the illusion of a living, morphing structure. I played with transparency and saturation of the circles against a dark background for a glowing effect. The user interaction is key: moving the mouse horizontally changes the symmetry, while vertical movement changes the rotation speeds.

Technical Details

  • State Management: Crucially used `push()` and `pop()` functions to isolate the transformations (translations, rotations) applied to the central kaleidoscope structure, ensuring the transformations didn't affect the rest of the canvas.
  • Interactive Symmetry: Mapped the `mouseX` position to control the degree of symmetry and pattern rotation, giving the user direct influence over the morphing illusion structure.
  • Pattern Generation: Employed nested `for` loops to efficiently draw the dense, repeating geometric patterns (circles/lines) necessary to create the visual complexity that drives the optical illusion effect.
