Assignment #2 — Face Generator

Project Overview

This assignment was a challenge in balancing randomness and control. The goal was to use procedural generation to create purposeful, evolving forms, ensuring slight adjustments to variables created deliberate variations rather than chaos. The sketch uses the random generate function to vary the shapes, sizes, and colors of the facial components every time the user clicks play.

Technical Details

  • Procedural Generation: Incorporated the `random()` function** throughout the sketch to control the size, color, and position of individual facial features, ensuring high variation upon each play.
  • Interactive Control: Used the `mousePressed()` function to trigger the generation of a new, unique face, balancing the sketch's internal randomness with direct user input.
  • Custom Features: Created custom functions to draw modular facial elements (eyes, nose, mouth), allowing those functions to accept different parameters (variables) for scale and color to manage the procedural output.
